# Dependencies for webnet_app

Install the following packages:

## Core

```bash
npm install firebase @react-navigation/native @react-navigation/stack react-native-screens react-native-safe-area-context
```

## Optional (for NativeWind / Tailwind v4)

```bash
npm install nativewind tailwindcss
```

## Summary

| Package | Purpose |
|---------|---------|
| `firebase` | Auth, Firestore, Storage |
| `@react-navigation/native` | Navigation core |
| `@react-navigation/stack` | Stack navigator |
| `react-native-screens` | Native screen containers |
| `react-native-safe-area-context` | Safe area handling |
| `nativewind` | Tailwind for React Native |
| `tailwindcss` | Tailwind v4 (if using NativeWind) |

## One-liner

```bash
npm install firebase @react-navigation/native @react-navigation/stack react-native-screens react-native-safe-area-context
```
